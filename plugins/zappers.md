# Zappers

 Zap up to / Zap to commands (to string / regexp)

 - - -
## Functions

### zap-to-regexp [command]

Zap text up to a regexp, `arg` can be minus to zap backwards.

<sup>function signature</sup>
```lisp
(zap-to-regexp (&optional arg))
```
