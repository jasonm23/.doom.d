# Diff region

 - - -
## Functions

### diff-region [command]

Select a region to compare.

<sup>function signature</sup>
```lisp
(diff-region)
```

- - -

### diff-region-now [command]

Compare current region with region already selected by `diff-region`.

<sup>function signature</sup>
```lisp
(diff-region-now)
```
